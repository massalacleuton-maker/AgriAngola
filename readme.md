# 🚜 AgriAngola

> **Plataforma digital para facilitar a distribuição de produtos agrícolas entre o campo e as cidades em Angola.**

---

## 📋 Sobre o Projeto
O **AgriAngola** é uma iniciativa tecnológica que visa conectar produtores rurais diretamente aos mercados urbanos. O objetivo é otimizar a cadeia logística agrícola de Angola, garantindo que os produtos cheguem frescos e com preços justos aos consumidores.

## ✨ Funcionalidades (Previstas)
*   **Interface Premium:** Design moderno com suporte a Dark Mode e Glassmorphism.
*   **Catálogo de Produtos:** Visualização clara das ofertas agrícolas das diferentes províncias.
*   **Painel do Produtor:** Área dedicada para gestão de estoque e vendas.
*   **Logística Inteligente:** Sistema para facilitar o transporte rural-urbano.

## 🛠️ Tecnologias Utilizadas
*   **HTML5 / CSS3:** Estrutura e estilização moderna.
*   **JavaScript:** Interatividade e lógica do sistema.
*   **Git & GitHub:** Controle de versão e hospedagem.
